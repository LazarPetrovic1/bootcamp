# Why QRadar SIEM?

### What Makes QRadar So Special - Part One

***Q-Radar*** collects logs.

Logs are kept in an *asset database*.

Logs don't convey much about the network, but *flows*.

Flows tell about:
+ Source IP
+ Destination IP
+ Port used
